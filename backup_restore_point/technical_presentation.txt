Technical Presentation
=====================

## Solution Overview
The recruitment system provides basic AI-powered candidate assessment and workflow management. The system includes candidate screening, interview scheduling, and status tracking.

## System Components
The system uses:
- Streamlit for frontend interface
- Python for backend processing
- Gemini AI for candidate assessment
- SMTP for email communications
- Calendly for interview scheduling

- User interface for input and display

- JSON for data storage
